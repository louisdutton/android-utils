# Privacy Policy

Wallet does not collect, sell, or transmit personal information.

To provide app functionality, Wallet may request access to:

- Camera: used locally to scan barcodes.
- NFC: used locally to pause NFC while showing a barcode, which can prevent payment or gate apps from triggering accidentally.
- Storage on Android 5 and 6 only: used to create or import backups on older Android versions.

Card data is stored locally on the device. Import, export, sharing, and backup
features only use the destinations the user explicitly chooses.

Wallet does not include Google Play crash reporting. The ACRA email crash
reporter from upstream Catima is not included in this fork.
